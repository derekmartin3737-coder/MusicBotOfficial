# Presentation Assets

This folder contains reusable slide/demo assets for explaining the terminal workflow.

- `render_terminal_annotation.ps1` and `.bat` annotate the "play newest MIDI" terminal screenshot.
- `render_song_selection_annotation.ps1` and `.bat` annotate the song-selection terminal screenshot.
- `terminal_annotated.png` and `song_selection_annotated.png` are rendered slide-ready images.
- `terminal_annotation_overlay.html` is an alternate markup-based overlay reference.

These files are presentation/support materials, not part of the runtime piano player.
