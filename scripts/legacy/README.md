This folder keeps older experiments and reference scripts that are not part of the main workflow.

Use `..\convert_midi.py` from the repository root for the supported MIDI-to-Arduino export flow.

Treat files in this folder as historical prototypes only. They helped prove out MIDI parsing and generated-header ideas, but they are not maintained for the PCA9685 solenoid runtime. If a classmate wants to run the current project, start with `scripts\play_piano.py`, `scripts\piano_tools.py`, and `arduino\MusicBotOfficial\MusicBotOfficial.ino` instead.
